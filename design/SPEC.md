## “心轮”微信小程序 - 核心功能设计开发文档 

### 1. 主视图：象限图与守护星

#### 1.1 UI规范

- **画布尺寸**：正方形绘图区域（推荐 600x600 rpx），周围预留坐标轴标签空间。整体画布居中于屏幕。
- **背景色**：暖浅灰或米白色 `#F7F5F2`，避免纯白刺激。
- **坐标轴**：
  - 线条颜色：`#3A4A5C`，宽度 2rpx，固定不动。
  - 横轴（X轴）：紧急性，向右箭头，标签文字“紧急度 →”。
  - 纵轴（Y轴）：重要性，向上箭头，标签文字“重要性 ↑”。
  - 标签字体颜色：`#8B9CAF`，大小 20rpx。
  - 中线固定，将象限均匀分为四部分。
- **象限底色**（透明度 0.15，仅作空间暗示）：
  - 第一象限（右上，紧急且重要）：淡暖橙 `rgba(255,200,180,0.15)`
  - 第二象限（左上，重要不紧急）：淡蓝绿 `rgba(180,220,210,0.15)`
  - 第三象限（左下，不重要不紧急）：淡灰 `rgba(200,200,200,0.15)`
  - 第四象限（右下，紧急不重要）：淡黄 `rgba(250,230,170,0.15)`
- **当前活跃焦点**：
  - 形状区分：
    - 待办事项：实心圆形，半径 18rpx。
    - 纯粹焦虑源：实心菱形（旋转45°正方形），边长等效 18rpx。
  - 颜色映射焦虑等级（1-5）：
    - 1级（平静）：`#A0C4D8` (宁静蓝)
    - 2级：`#8BB0C4`
    - 3级：`#F0C27A` (琥珀黄)
    - 4级：`#E89F6E`
    - 5级（极度焦虑）：`#F08A7E` (珊瑚粉)
  - 若焦点有过版本变动（即历史版本>1个），其图标右上角叠加一个微小的“幽灵尾巴”标识：弧形线条，大小 8rpx，颜色 `#FFD166`。
- **聚合气泡**：
  - 触发条件：任一象限内活跃焦点数量 > 5 个。
  - 样式：以该象限几何中心为圆心，绘制实心圆，半径 = 25 + (焦点数 × 2) rpx。
  - 填充色：`#FFD166`，数字颜色白色，加粗居中。
  - 不显示单个焦点，仅显示聚合数字。
- **守护星背景层**：
  - 数据源：所有状态为“已化解”的焦点。
  - 样式：半透明暖金色星形（五角星），颜色 `rgba(255,215,0,0.4)`，大小 6rpx，随机旋转微小角度。
  - 位置：依据该焦点化解时的紧急性/重要性坐标落点。
  - 层级：绘制在坐标轴和象限底色之上，但在活跃焦点之下，不遮挡当前焦点。
  - 交互：纯背景装饰，不响应任何点击或触碰。
- **固定入口**：
  - 右上角奖杯图标（已化解博物馆入口），大小 40rpx，颜色 `#FFD166`，点击进入博物馆页。
  - 底部中央悬浮“+”按钮，圆形，直径 80rpx，背景 `#FFD166`，白色加号图标，带柔和阴影。点击触发捕捉流程。

#### 1.2 交互定义

- **焦点点击**：手指点击活跃焦点（命中区域半径 25rpx 内），跳转至“焦点详情页”，同时传入该焦点 ID。
- **气泡点击**：点击聚合气泡，弹出居中半透明浮层，浮层内纵向滚动列出该象限所有焦点标题。点击任一标题进入详情，点击浮层外部或“关闭”按钮关闭浮层。
- **长按操作**：无额外操作。
- **主图手势**：禁止双指缩放或拖动，保持坐标系固定。未来可扩展左右横扫切换“全局对比”时间快照（见5）。
- **数据加载状态**：画布在数据未完成加载前，显示极简的骨架屏（坐标轴线条 + 空白象限底色），无转圈加载动效，避免加剧焦虑感。
- **空状态**：若无任何活跃焦点，象限图显示居中淡色文字“捕捉你的第一个想法，让它落在纸上”，颜色 `#C0C6CC`，下方指引箭头指向“+”按钮。

### 2. 捕捉与评分向导

#### 2.1 UI规范

- **呈现方式**：建议使用半屏弹窗（微信 `page-container` 组件），或独立全屏页（更专注），背景保持柔和。
- **标题输入**：顶部自动聚焦的 `<input>` 框，占位符文字：“记录一个任务或担忧…”，无边框底部横线样式，输入文字颜色 `#2C3E50`。
- **类型切换**：标题下方两个选项卡片，左为“待办事项 ○”，右为“焦虑源 ◇”，选中态有温和背景高亮（`#FFD166` 透明度 0.2）。默认选中“焦虑源”。
- **评分滑块区域**：
  - 三条独立滑杆，左侧为属性名（紧急性、重要性、焦虑等级），右侧实时显示当前数值。
  - 滑杆样式：轨道色 `#E5E9F0`，已选中部分颜色对应属性（紧急性用橙系，重要性用蓝系，焦虑等级用珊瑚系），滑块圆点白色带阴影。
  - 紧急性：1 “不急” ～ 5 “紧急”。
  - 重要性：1 “不重要” ～ 5 “重要”。
  - 焦虑等级：1 “平静” ～ 5 “极度焦虑”，额外在滑杆下方附一条由宁静蓝渐变至珊瑚粉的细条示意。
  - 默认值：所有滑块初始置为 3。
- **心理学引导气泡**：
  - 位置：在滑块区域下方，根据当前紧急性与重要性的组合动态显示。
  - 样式：柔和的圆角矩形，背景 `#FFF9E6`，左侧带一条 4rpx 宽的 `#FFD166` 竖线，文字 `#3A4A5C`，字号 24rpx。
  - 过渡动画：气泡淡入显示，停留 5 秒后自动淡出，也可手动点击“知道了”关闭。非强制，不阻塞操作。
  - 引导内容映射：
    - (≤2, ≤2) 不紧急不重要：_“放下这个担忧，会为你此刻的身心腾出多少空间？”_
    - (≥4, ≤2) 紧急不重要：_“这件事如果拒绝或交给别人，最坏的结果是什么？”_
    - (≤2, ≥4) 重要不紧急：_“能让这件事推进的最小一步是什么？”_
    - (≥4, ≥4) 紧急重要：_“深呼吸，专注完成它。完成后记得来庆祝。”_
    - 其他中间组合：不弹出，保持界面安静。
- **确认按钮**：底部居中“确认添加”，圆角按钮，背景 `#FFD166`，文字白色加粗。点击后保存。

#### 2.2 交互细节

- 用户滑动任一评分滑块时，实时计算所属象限，更新引导气泡内容（如需要）。
- 点击“确认添加”后：
  1. 若标题为空，按钮不可点击，给出轻微提示（输入框抖动或边框变红）。
  2. 有效则执行添加，关闭弹窗，返回主图。
  3. 主图绘制时，新增焦点可伴随一个微小的飞入动画（从中心飞向对应坐标），强化反馈。
- 取消添加：点击弹窗外部或左上角关闭按钮，不保存任何内容。

### 3. 焦点详情与版本时间轴播放器

#### 3.1 UI结构

- **页面布局**：导航栏显示焦点标题。页面分为上下两部分。
- **上部：演化小图区**
  - 占据约 40% 高度，正方形 Canvas (350x350 rpx)。
  - 仅显示本焦点的移动轨迹：折线（或平滑贝塞尔曲线），颜色 `#FFD166`，宽度 2rpx，末端带微小箭头。
  - 路径上根据版本数量绘制节点圆点（直径 10rpx），当前选中版本节点高亮（填充白色，外环 `#FFD166`）。
  - 背景显示极淡的象限网格，帮助理解位置。
- **下部：时间轴与当前信息**
  - **时间轴**：水平滑动区域，高度 80rpx。刻度线上按时间顺序等距排列版本节点（小圆点）。选中节点放大并改变颜色。上方显示选中版本的日期与时间（格式：MM/DD HH:mm）。
  - **版本信息卡片**：圆角卡片，展示选中版本的紧急性、重要性、焦虑等级数值，以及“状态”（进行中/已化解）。
  - **修改原因记录区**：若该版本记录有 `changeNote`，卡片下方显示一个对话气泡样式的文本块：“当时我想：xxxx”。字体略小，颜色 `#5A6B7F`。
  - 若焦点已化解，在标题旁显示化解时间标签。
- **操作按钮**（固定底部）：
  - “编辑评分”（仅活跃焦点可见）：点击后当前版本滑块变为可调，调整后自动新增版本并弹出记录原因框。
  - “标记为已化解”（活跃焦点可见）：暖金色按钮，点击触发仪式并改变状态。
  - “删除焦点”（文字链接，置于角落，需二次确认）。

#### 3.2 交互流程

- **拖动时间轴**：手指在时间轴区域水平滑动，选中节点变化，上方演化图的高亮节点和路径位置实时跟随变化。回馈流畅无延迟。
- **切换版本查看记录**：点击时间轴上的任意节点，即可选中该版本，信息卡片与感受记录同步切换。
- **编辑评分**：进入编辑态后，紧急性与重要性滑块变为可调节，滑动时演化图上的点实时移动预览。确认后，自动将新评分保存为新版本，并在下方弹出轻量文本域：“是什么让你的认知改变了？(可选)”，用户填写后保存。版本时间轴新增一个节点。
- **化解仪式**：点击“标记为已化解”并确认后：
  1. 演化图中的焦点图标执行缩小并变色为暖金色的动画。
  2. 触发手机短震 (`vibrateShort`)。
  3. 动画完毕后自动返回主象限图，新的守护星已添加至背景层。

### 4. 已化解博物馆

#### 4.1 页面入口与布局

- **入口**：主象限图右上角奖杯图标。
- **列表样式**：垂直滚动，每项一张卡片。
- **卡片内容**：
  - 化解日期（左侧，着重显示）。
  - 焦点标题（主文字）。
  - 微型象限坐标图（24x24rpx），示意化解时的紧急/重要位置。
  - 成就标签（彩色小徽章），例如：“风暴平息者”、“边界守护者”、“远见家”、“卸载达人”。标签颜色温和，与象限色系呼应。
- **空状态**：若尚无已化解焦点，显示“这里还空着。每一个被你安然放下的担忧，都会化作这里的星星。”配以一颗小星图标。

#### 4.2 交互

- 点击任意卡片，进入该焦点**只读详情页**（复用详情页布局，但不可编辑评分或再次化解，时间轴可拖动查看历史）。
- 博物馆页面下拉可刷新。
- 长按卡片无额外操作。

### 5. 全局对比与认知摘要

#### 5.1 全局对比模式

- **开启方式**：在主象限图顶部增加时间筛选栏，选项：“此刻”、“1周前”、“1月前”、“自定义日期”。默认“此刻”。
- **表现**：选取历史时间点后，象限图重新绘制，展示该时间断面下的活跃焦点分布（即取各焦点在该时间点之前的最新版本状态）。已化解的守护星也仅显示该时间前已化解的星点。
- **幽灵叠加**（可选增强）：提供一个“对比”开关，开启后，以半透明方式同时显示当前分布与历史分布，颜色略作区分，直观看出焦虑地形的变迁。
- **交互**：在历史视图中，焦点不可点击或编辑，仅用于观察。点击空白区可退出历史视图回到“此刻”。

#### 5.2 认知摘要

- **生成频率**：每周一次自动总结，也可在博物馆页面顶部点击“本周摘要”手动生成。
- **展示位置**：可在博物馆页面固定区域，或以卡片形式弹出。
- **摘要内容模板**（基于数据统计）：
  - 本周新增焦点数、已化解数。
  - 从高焦虑(≥4)降至低焦虑(≤2)的焦点数量。
  - 被卸载的“紧急不重要”事件数。
  - 引用用户记录的 `changeNote` 中具有代表性的关键词，如“拒绝也没关系”、“最坏结果可控”。
  - 平均焦虑等级变化趋势（↓或↑）。
  - 温和的结语：“你在主动梳理内心，这本身就很了不起。”
- **数据来源**：后端对焦点集合按时间过滤版本和状态变化进行计算。

### 核心数据字段参考

（此部分用于数据库设计，非UI，保留以明确数据关系）

**集合名：focuses**

| 字段 | 类型 | 说明 |
|------|------|------|
| _id | String | 自动生成 |
| title | String | 焦点标题 |
| type | String | "task" 或 "anxiety" |
| status | String | "active" 或 "resolved" |
| createdTime | Date | 创建时间 |
| resolvedTime | Date | 化解时间（status为resolved时有值） |
| versions | Array | 版本历史，每个对象包含： |
|  |  | - timestamp: Date |
|  |  | - urgency: Number (1-5) |
|  |  | - importance: Number (1-5) |
|  |  | - anxietyLevel: Number (1-5) |
|  |  | - changeNote: String (可空) |
